How to use pyinstaller for GUI windows apps
run the command below in the py folder in Powershell
replace main.py with any py file
pyinstaller --onefile --windowed --clean main.py

For the web app used stream
To install:
pip install streamlit

to run the server:
streamlit run web.py
streamlit hello
