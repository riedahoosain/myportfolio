Dictionary API

Returns the meaning of the word provided using format below:

URL format: http://127.0.0.1:5000/api/v1/word


URL Example: http://127.0.0.1:5000/api/v1/sun


URL Example: http://127.0.0.1:5000/api/v1/moon


The samples above would return:

{ "definition": "1. Any star, especially when seen as the centre of any single solar system.\n2. The particular star at the centre of our solar system, from which the Earth gets light and heat.\n", "word": "sun" }


Warning Running the below loads the entire Dictionary
